# tdg5 Thor Generators

Thor generators for projects and miscellanea.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'tdg5-thor-generators'
```

And then execute:

```bash
bundle
```

Or install it yourself as:

```bash
gem install tdg5-thor-generators
```
